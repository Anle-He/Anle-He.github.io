# 个人资料填写清单

发布前请确认：

- 中英文姓名、身份、简介、所在地和邮箱均已替换。
- GitHub、Google Scholar、博客和源码仓库链接均属于你本人。
- 头像、论文图、机构 Logo 已获得公开展示许可。
- 论文作者顺序、发表状态、链接和年份准确。
- 未包含家庭住址、证件号、私人电话、未公开项目或内部文件。
- `npm run validate` 不再提示占位符。
- `npm run lint`、`npm run audit`、`npm run build` 全部通过。

主要编辑入口是 `content/site.json` 和 `content/zh/site.json`，其他履历内容
位于 `content/` 及 `content/zh/` 的对应 JSON 或 Markdown 文件中。
